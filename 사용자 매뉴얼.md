# Edu-Calc AI 사용자 매뉴얼 (User Manual)

**Edu-Calc AI**는 단순한 정답 제공을 넘어, 사용자 수준별 맞춤형 설명과 핵심 개념 학습을 돕는 'AI 기반 단계별 학습형 공학계산기'입니다.

---

## 🚀 1. 빠른 시작 가이드 (Execution Guide)

본 프로그램은 **백엔드(Flask)** 서버와 **프론트엔드(HTML/JS)**로 구성되어 있습니다.

### 1.1 사전 준비 사항
- **Python 3.10** 이상의 버전이 설치되어 있어야 합니다.
- **Groq API Key**가 필요합니다. ([console.groq.com](https://console.groq.com/)에서 발급 가능)

### 1.2 서버 실행 방법
1.  **의존성 설치**: 터미널에서 프로젝트 루트 디렉토리로 이동 후 아래 명령어를 실행합니다.
    ```bash
    pip install -r api/requirements.txt
    ```
2.  **환경 변수 설정**: `.env.example` 파일을 복사하여 `.env` 파일을 생성하고, 발급받은 API 키를 입력합니다.
    ```text
    GROQ_API_KEY=your_actual_api_key_here
    ```
3.  **백엔드 서버 실행**:
    ```bash
    python api/index.py
    ```
    - 서버가 `http://127.0.0.1:5001`에서 실행됩니다.

### 1.3 클라이언트 접속
- 프로젝트 루트의 **`index.html`** 파일을 웹 브라우저(Chrome, Edge 권장)로 직접 엽니다.

### 1.4 외부(동일 네트워크) 접속 방법
같은 Wi-Fi나 네트워크를 사용하는 다른 기기(스마트폰, 다른 노트북 등)에서 접속하려면:
1.  **서버 실행 기기**의 IP 주소를 확인합니다. (예: `192.168.0.144`)
2.  **다른 기기**의 브라우저에서 아래 주소를 입력합니다:
    ```text
    http://192.168.0.144:5001
    ```
    - 주의: 방화벽 설정에 따라 5001번 포트 허용이 필요할 수 있습니다.

## 🖥️ 2. 주요 기능 사용법

### 2.1 학습 레벨 선택
화면 상단의 레벨 셀렉터에서 본인의 수준에 맞는 단계를 선택합니다.
- **초등**: 아주 쉬운 단어와 사물 비유를 통한 설명.
- **중등**: 교과 과정 용어를 사용한 논리적 풀이.
- **고등**: 심화 개념과 수식의 유도 과정을 포함한 전문 튜터링.
- **대학**: 학술적 정의와 증명을 포함한 고난도 공학 풀이.

### 2.2 수식 및 질문 입력
입력창에 계산하고 싶은 수식이나 수학적 질문을 자연어로 입력합니다.
- 예: `2x + 10 = 30 풀어줘`, `피타고라스의 정리가 뭐야?`, `sin(x)의 미분법 알려줘`

### 2.3 결과 카드 확인
AI가 분석을 완료하면 다음과 같은 4가지 카드가 순차적으로 나타납니다.
1.  **결과 (Result)**: 최종 계산 값이나 정답을 확인합니다.
2.  **핵심 개념 (Core Concept)**: 문제 풀이의 중심이 되는 원리를 한눈에 파악합니다.
3.  **상세 풀이 (Detailed Explanation)**: '상세 풀이'를 클릭하여 단계별 계산 과정을 학습합니다.
4.  **유사 문제 (Practice Problem)**: 제시된 문제를 직접 풀어보며 실력을 점검합니다.

## 🛠️ 3. 트러블슈팅 (Q&A)

**Q: 화면에 결과가 나오지 않고 에러 메시지가 뜹니다.**
- **A**: 백엔드 Python 서버(`index.py`)가 정상적으로 실행 중인지 확인해 주세요. 터미널에 `http://127.0.0.1:5001` 문구가 떠 있어야 합니다.

**Q: 수식이 깨져서 보입니다.**
- **A**: 인터넷 연결 상태를 확인해 주세요. 수식 렌더링에 필요한 `KaTeX` 라이브러리를 외부 CDN에서 불러오기 때문입니다.

**Q: API 키가 잘못되었다고 합니다.**
- **A**: `.env` 파일의 `GROQ_API_KEY` 값에 공백이나 오타가 없는지, 발급받은 키가 유효한지 확인해 주세요.
