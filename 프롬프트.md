# Role
너는 25년 경력의 숙련된 Full-stack Software Architect이자 IT 기술 작가야. 
제공된 아이디어를 바탕으로 개발팀과 이해관계자가 즉시 이해할 수 있는 수준의 '프로그램 개요서(Program Specification)'를 Markdown 형식으로 작성해줘.

# Project Overview
- 프로젝트명: AI 기반 단계별 학습형 공학계산기 (가칭: Edu-Calc AI)
- 핵심 가치: 단순 결과 도출을 넘어, 사용자 수준(초/중/고/대)별 핵심 개념 정리 및 유사 문제 제공을 통한 학습 효과 극대화.

# Technical Stack
- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Backend: Python Flask
- AI Engine: Groq AI API (Llama 3 혹은 Mixtral 모델 활용)
- Deployment: Git Repository 연동 및 Vercel 배포

# Required Sections in Document
다음 항목들을 포함하여 상세하게 작성해줘:

1. **프로젝트 개요**: 프로그램의 목적과 기대 효과.
2. **주요 기능 (Feature List)**:
   - 자연어 기반 수식 입력 및 변환 처리.
   - 4단계 난이도별(초등/중등/고등/대학) 맞춤형 설명 기능.
   - 핵심 개념 요약 및 유사 문제 생성 로직.
3. **시스템 아키텍처**:
   - Frontend(HTML/JS) <-> Backend(Flask) <-> AI(Groq API) 간의 데이터 흐름 설명.
4. **UI/UX 설계 방향**: 
   - 사용자가 '편한 말투'로 입력할 수 있는 인터페이스 및 결과 시각화 방안.
5. **데이터 스키마 및 프롬프트 엔지니어링 전략**:
   - Groq AI에게 난이도별 답변을 유도하기 위한 시스템 프롬프트 설계 전략.
6. **배포 및 운영 계획**: Git과 Vercel을 활용한 CI/CD 구성안.
